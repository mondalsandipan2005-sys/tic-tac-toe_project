# tic-tac-toe_project
Tic Tac Toe 🎮
<br>
<br>
A simple and responsive Tic Tac Toe Game built using HTML, CSS, and JavaScript.
This project demonstrates DOM manipulation, event handling, game logic implementation, and responsive frontend design.
<br>
<br>
🚀 Features
<br>
✅ Two-player gameplay (X vs O)
<br>
✅ Interactive UI
<br>
✅ Win detection logic
<br>
✅ Draw detection
<br>
✅ Reset game functionality
<br>
✅ Responsive design using vmin
<br>
✅ Dynamic DOM manipulation
<br>
✅ Clean and beginner-friendly code
<br>
<br>

🛠️ Technologies Used
<br>
HTML5
<br>
CSS3
<br>
JavaScript
<br>
<br>
🎯 How the Game Works
<br>
<ol><li>Player X starts the game.</li>
<li>Players take turns clicking on the boxes.</li>
<li>The game checks for:</li>
Row win
<br>
Column win
<br>
Diagonal win
<br>
<li>If all boxes are filled without a winner, the game declares a draw.</li>
<li>The Reset button restarts the game.</li>
</ol>
